Changing map location

You have to change latitude and longtitude value of map data attribute . If you dont know how to get latitude and longitude value search on google or send us your location in google map, we will do that for you. here is a screenshot that shows where exactly you have to change these values. 

You have customize the data attribute of your google map  See the image below 

The Required attribute is

  • unique id
  • data-map-lat
  • data-map-lng
  • data-map-zoom
